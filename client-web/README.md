# client-web

用户端 AI 助手前端，基于 Next.js App Router。

## 启动说明

```bash
npm install
npm run dev
```

默认启动地址：

- `client-web`: `http://localhost:3001`
- `admin-web`: `http://localhost:3000`

## 常用命令

```bash
npm run dev
npm run build
npm run start
npm run lint
```

## 接口说明

前端默认请求：

- `NEXT_PUBLIC_API_BASE_URL`
- 默认值：`http://localhost:8080`

如需修改后端地址，可在启动前设置环境变量。
